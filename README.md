# K-English Trainer — Kearney流 英会話トレーナー

Kearney（A.T. カーニー）が公開している連載「**Kearney流英語力習得術**」（Kearney Japan Magazine・全 4 回）の学習理論を、
**実際に手を動かして練習できる機能**へ落とし込んだ、ブラウザ完結型の英会話学習 Web アプリです。

- 外部 API・有料サービス・ビルド不要。`index.html` を開くだけで動きます
- 学習の進捗はブラウザ内（localStorage）に記録され、ダッシュボードで可視化されます
- PC・スマートフォンの両方に対応（レスポンシブ）／ライト・ダークモード自動対応
- UI は日本語、学習コンテンツはビジネス英語（コンサルティングの現場を想定）

## 2. 原典のメソッドと機能への落とし込み

### 中核フレームワーク：サバイバルトライアングル（vol.2）

英語ノンネイティブが実践の中で英語を習得するための 3 要素。本アプリの全機能はこの 3 要素に対応します。

```
            Mindset（考え方）
              ▲
             ／ ＼      → Survival（実践での生存）
            ／   ＼     → Confidence & Skills（自信とスキル）
   Method ▲───────▲ Technique
  （学び方）        （実践での使い方）
```

当面のゴールは「英語を使って一定のプロジェクトワークをこなせるスキル」＋「英語でのワーク・コミュニケーションに壁を感じないマインド」。

### 対応表

| 原典の要素 | 内容（要約） | アプリでの機能 |
|---|---|---|
| **Mindset ①** 魚は逃さないマインド | 英語を使う機会には Yes と言ってとにかく飛び込む。「やってみます」「できます」とまず言う | 🏠 ホームの**日替わりマインドセットカード**（事例のエッセンス付き） |
| **Mindset ②** ミスしたっていいマインド | 完璧な英語という理想を捨て、ミスを恐れず数を打つ。聞き返し・言い換えは当たり前 | 同上＋全練習が減点式でなく「挑戦回数」を積む設計 |
| **Mindset ③** 憧れを忘れないマインド | 英語で何をしたいかをビビッドに思い描き、苦しいときの燃料にする | 🧭 診断の**憧れメモ**（ホームに常時表示） |
| **Method: リーディング** | 簡単な英語をざっくり速く読む。1 日数千〜1 万語 | 📖 **速読リーディング**：WPM 計測＋理解度クイズ（パッセージ 5 本） |
| **Method: リスニング** | 200 ワード程度の短いパッセージを、そらんじられるまでシャドーイング。1 文ずつ→全体へ | 🎧 **シャドーイング**：5 ステップ手順・1 文ループ・速度調整・スクリプト非表示・録音確認（5 本） |
| **Method: ライティング** | パターンごとに表現をストックし、実際に例文を書いてみる | 📚 **型・表現**：6 シーン 48 表現の SRS ＋**表現ノート**（自分の例文を書いて登録） |
| **Method: スピーキング** | 毎日話す機会を持ち、覚えた表現を「決めた回数」意識して使う。三技能がベース | ⚡ **瞬間英作文**（10 秒制限・SRS・72 文）＋ 🎭 **ロールプレイ**（4 シナリオ） |

## 3. 起動方法

ビルドもサーバーも不要です。

1. このリポジトリをダウンロード（`git clone` または ZIP）
2. **`index.html` をブラウザで開く**（ダブルクリックで OK）

ローカルサーバー経由で開きたい場合は：

```bash
cd Kearney-English
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

### 動作環境について

- 推奨ブラウザ：**Chrome / Edge**（音声合成＋音声認識の両方が動作）。Safari / Firefox は音声合成のみ
- 英文の読み上げはブラウザ内蔵の音声合成（Web Speech API）を使用 — **API キー・課金不要**
- 🎤 音声認識・録音は対応ブラウザのみ表示される任意機能で、非対応でも全メニューを自己評価方式で学習できます
- 学習データはすべてブラウザの localStorage に保存（サーバー送信なし）。「記録・振り返り」画面からエクスポート／インポート可能

## 5. 技術方針

- **Vanilla HTML/CSS/JavaScript のみ**（フレームワーク・ビルドツール・CDN 依存なし）。`file://` で直接開いても動くよう、ES Modules ではなく通常のスクリプト読み込みを採用
- **データ保存はブラウザ内で完結**：localStorage（プラン／SRS 状態／学習ログ／表現ノート／振り返り）
- **音声は Web Speech API**：外部 TTS・音声認識サービスを一切使わない
- **チャートは自前 SVG 描画**：CSS カスタムプロパティ参照でダークモードに自動追従。色は色覚多様性に配慮して検証済みのパレットを使用
- 学習コンテンツ（英文・対訳・解説）はすべて本リポジトリ内に同梱

## 6. 自動コンテンツ配信（運用）

コンテンツは Claude のスケジュール実行（ルーチン）により自動更新される。**鮮度型は毎日、反復型（SRS・シャドーイング）は消化ペースに合わせて週次**という設計。

| ルーチン | 頻度 | 更新内容 | 書き込み先 |
|---|---|---|---|
| デイリー配信 | 毎朝 6:00 JST | 時事トピックのオリジナル英文 5 本（ジャンル別・クイズ付き）。直近 7 日分を保持 | `js/data/daily.js`（上書き） |
| ウィークリー配信 | 毎週日曜 21:00 JST | 瞬間英作文 +12 文／シャドーイング +1 本／PREP 時事お題 +3 題（隔週で表現 +8、月初のみロールプレイ +1） | `js/data/extra.js`（追記） |

- ルーチンは**配信専用ファイルのみ**変更し、手書きの基本データには触れない
- ニュースは「トピックの参照」のみで本文の転載はしない（全パッセージはオリジナル書き下ろし）
- 追加が蓄積しても UI は「今日やるべきものだけ」を表示する：SRS は復習期限のカード＋新規（1 日 8 枚まで）を自動選択、シャドーイングは「今週の 1 本」、リーディングは「今日の 5 本」。過去分はアーカイブ（折りたたみ）へ

## 7. 使い方（推奨フロー）

1. 初回アクセス時に「🧭 診断・プラン」で目的（活用シーン）・5 スキル自己評価・学習時間・憧れメモを入力 → 1 日メニューが生成される
2. 毎日「🏠 ホーム」の今日のメニューを上から実施（実施時間は自動記録され、チェックがついていく）
3. 会議やドラマ・業務で出会った表現は「📚 型・表現 → 表現ノート」に**自分の例文つきで**追加
4. 週末に「📈 記録・振り返り」で KPT を記入し、必要なら「🧭 診断・プラン」で自己評価とプランを更新する
