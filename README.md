# K-English Trainer — Kearney流 英会話トレーナー

Kearney（A.T. カーニー）が公開している「Kearney流英語力習得術」シリーズの考え方をもとに、
その学習理論を**実際に手を動かして練習できる機能**へ落とし込んだ、ブラウザ完結型の英会話学習 Web アプリです。

- 外部 API・有料サービス・ビルド不要。`index.html` を開くだけで動きます
- 学習の進捗はブラウザ内（localStorage）に記録され、ダッシュボードで可視化されます
- PC・スマートフォンの両方に対応（レスポンシブ）／ライト・ダークモード自動対応
- UI は日本語、学習コンテンツはビジネス英語（コンサルティングの現場を想定）

## 2. 再構成した「Kearney流」5 原則と機能への落とし込み

| # | 原則(再構成したメソッド) | アプリでの機能 |
|---|---|---|
| 1 | **イシュードリブン** — なんとなく勉強しない。英語を使う目的から逆算して弱点を特定し、限られた時間を最も効く練習に配分する | 🧭 **診断・学習プラン**：目的（会議／電話会議／VERSANT 対策／海外協働）と 5 軸自己評価・学習可能時間から、1 日の推奨メニューを自動設計。配分ロジックは「目的への効き目 × 弱点ブースト」 |
| 2 | **型のストック** — 会議・電話・雑談の英語は「その場で作文」せず、頻出の型を大量に覚えてそのまま出す | 📚 **型・表現トレーニング**：6 シーン 48 表現＋自分で追加できる「表現ノート」。日本語→英語の想起テストを**間隔反復（SRS）**で回して定着させる |
| 3 | **瞬発力（自動化）** — 知っている英語を「10 秒以内に口から出る英語」に変える。VERSANT の文章構文・語彙に直結 | ⚡ **瞬間英作文**：10 秒カウントダウン付きで日本語→英語を口頭変換。3 デッキ 72 文（基礎／会議・議論／数字・分析）。自己評価が SRS に反映され、忘れる直前に再出題 |
| 4 | **構造で話す** — 結論ファースト（Point→Reason→Example→Point）。構造は語学力を補う最強の武器 | 🏗️ **PREP 構成ドリル**：90 秒スピーキング→型に沿って書き出し→模範解答と比較→セルフチェック 5 項目。お題 12 問 |
| 5 | **耳と口を実戦速度に** — 聞き流さない。1 本のスクリプトを完全に口が回るまで仕上げ、実践の場で使う | 🎧 **シャドーイング**（5 ステップ手順・速度調整・1 文ループ・スクリプト非表示・録音自己確認）＋ 🎭 **実践ロールプレイ**（海外オフィスとの初回コール／有識者インタビュー／週次報告／質疑応答 — vol.3 の「英語活用実態」が扱う現場場面を再現） |
| ＋ | **記録と PDCA** — 学習を可視化し、週次で学び方自体を改善する | 📈 **記録・振り返り**：全セッション自動記録。連続日数・ヒートマップ・スキルレーダー・メニュー別累計をダッシュボードに表示。週次 KPT（Keep/Problem/Try）振り返り |

## 3. 起動方法

ビルドもサーバーも不要です。

1. このリポジトリをダウンロード（`git clone` または ZIP）
2. **`index.html` をブラウザで開く**（ダブルクリックで OK）

ローカルサーバー経由で開きたい場合は：

```bash
cd Kearney-English
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

### 動作環境について

- 推奨ブラウザ：**Chrome / Edge**（音声合成＋音声認識の両方が動作）。Safari / Firefox は音声合成のみ
- 英文の読み上げはブラウザ内蔵の音声合成（Web Speech API）を使用 — **API キー・課金不要**
- 🎤 音声認識・録音は対応ブラウザのみ表示される任意機能で、非対応でも全メニューを自己評価方式で学習できます
- 学習データはすべてブラウザの localStorage に保存（サーバー送信なし）。「記録・振り返り」画面からエクスポート／インポート可能

## 5. 技術方針

- **Vanilla HTML/CSS/JavaScript のみ**（フレームワーク・ビルドツール・CDN 依存なし）。`file://` で直接開いても動くよう、ES Modules ではなく通常のスクリプト読み込みを採用
- **データ保存はブラウザ内で完結**：localStorage（プラン／SRS 状態／学習ログ／表現ノート／振り返り）
- **音声は Web Speech API**：外部 TTS・音声認識サービスを一切使わない
- **チャートは自前 SVG 描画**：CSS カスタムプロパティ参照でダークモードに自動追従。色は色覚多様性に配慮して検証済みのパレットを使用
- 学習コンテンツ（英文・対訳・解説）はすべて本リポジトリ内に同梱

## 6. 使い方（推奨フロー）

1. 初回アクセス時に「🧭 診断・プラン」で目的・自己評価・学習時間を入力 → 1 日メニューが生成される
2. 毎日「🏠 ホーム」の今日のメニューを上から実施（実施時間は自動記録され、チェックがついていく）
3. 会議やオンライン英会話で出会った表現は「📚 型・表現 → 表現ノート」に追加（自分だけの復習カードになる）
4. 週末に「📈 記録・振り返り」で KPT を記入し、必要なら「🧭 診断・プラン」で自己評価とプランを更新する
